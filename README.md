# MVVM API Pagination Kingfisher Demo

## Overview
This project demonstrates an iOS application built using **SwiftUI + MVVM architecture** with API integration, pagination, and image loading using Kingfisher.

## Features
- MVVM architecture
- API integration
- Pagination support
- Lazy loading
- Image caching using Kingfisher

## Architecture
This project follows MVVM for clean separation of concerns:
- Model → Data layer
- View → UI
- ViewModel → Business logic

## Tech Stack
- Swift
- SwiftUI
- Kingfisher
- REST API
